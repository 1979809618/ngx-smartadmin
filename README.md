# ngx-smartadmin
The angular components for smart-admin.

# Usage

```bash
# Install package
npm i ngx-smartadmin -D

# Use In Typescript
import { SmartAdminModule } from 'ngx-smartadmin';

@NgModule({
  imports: [
    SmartAdminModule
  ]
})
export class AppModule {

}
```

# How to develop?

```bash
git clone 

# install deps
npm i 

# run dev
npm run dev

# build demo
npm run build

# build aot demo
npm run build:aot

# build publish package
npm run lib
```
